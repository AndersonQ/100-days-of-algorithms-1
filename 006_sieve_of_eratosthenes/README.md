Counting Primes (Sieve of Eratosthenes)
=======================================

Counting the prime numbers until N using the
[Sieve of Eratosthenes](https://en.m.wikipedia.org/wiki/Sieve_of_Eratosthenes) algorithm

Build
-----

```
$ go build
```

Run
---

```
$ ./006_sieve_of_eratosthenes -n=5
```
